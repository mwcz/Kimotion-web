Kimotion

WebGL-based interactive art exhibit framework. Make a kimotion!

Download as .zip Download as .tar.gz View source on GitHub

What is Kimotion?

Kimotion is a framework for building interactive art exhibits. With a pinch of JavaScript, Kinect, WebGL, and WebSockets you can create awesome interactive visualizations.
